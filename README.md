# 五柱十字

一个可直接部署到 GitHub Pages 的静态小项目。输入精确到分钟的出生时间，页面会生成年、月、日、时、分五柱，共十个干支字，并统计金木水火土的分布；同时向前后逐分钟寻找距离此生辰最近的五个“完美十字”。

## 规则

- 年柱：以立春为岁首，1984 年立春后为甲子年。
- 月柱：以节气分月，立春起寅月，惊蛰起卯月，依次顺排。
- 日柱：以 1912-02-18 为甲子日锚点顺推，默认启用 23:00 子初换日。
- 时柱：按十二时辰排支，以日干推出时干。
- 分柱：本项目的现代扩展。每小时 00 分为甲子，01 分为乙丑，逐分钟顺排，59 分为癸亥。
- 五行：每个天干和地支各记 1 点，总计 10 点。金木水火土各 2 点即为“完美十字”。

## 本地打开

直接用浏览器打开 `index.html`。

## 发布到 GitHub Pages

1. 推送本仓库到 GitHub 的公开仓库。
2. 打开仓库的 `Settings -> Pages`。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/ (root)`。

这是纯静态页面，不需要安装依赖或构建步骤。

## 说明

传统八字只有年、月、日、时四柱。“分柱”和“完美十字”是本项目为了把 60 分钟映射到六十甲子而设计的扩展规则，不是传统命理通行算法。
